# 美术工作流指南：Figma → PSD → FairyGUI

面向美术同学。按本文命名，程序可用工具转成 FairyGUI。

完整技术表见：[NAMING_RULES.md](./NAMING_RULES.md)

---

## 1. 流程一眼看懂

```
参考图 / 需求
  → Figma 按规则分层命名（可用 Cursor Agent 出草模）
  → 导出 PSD（文字必须还是文字）
  → psd2fgui 转换
  → FairyGUI 微调（列表、动效）
  → 程序接逻辑
```

**记住一句：图是皮，字是数，能点就必须叫 Btn_。工具只认名字，不认长得像不像按钮。**

---

## 2. 名字怎么起（最重要）

### 2.1 对照表（与项目一致）

| 这是什么 | 名字前缀 | 例子 |
|----------|----------|------|
| 文本（金币、等级…） | `Text_` | `Text_Gold`、`Text_Level` |
| 可换的图 | `Loader_` | `Loader_EquipIcon` |
| 列表区域 | `List_` | `List_Bag`（转完后在编辑器改成列表） |
| 进度条 | `Bar_` | `Bar_VipExp` |
| **按钮（能点）** | `Btn_` | `Btn_EquipSlot`、`Btn_Forge` |
| 普通组件块 | `Com_` | `Com_CurrencyItem` |
| 标签块 | `Label_` | `Label_Power` |
| 滑动条 | `Slider_` | `Slider_Volume` |
| 只是文件夹收纳 | 随便起 | `Header`、`BottomNav`（会保留层级） |

### 2.2 千万别用错

| 错误 | 正确 |
|------|------|
| 装备格叫 `Com_EquipSlot`（其实能点） | `Btn_EquipSlot` |
| 会变的字画在图上 | 拆成 `Text_` 文本层 |
| 按钮上的字没有 `@title`，程序改不了标题 | 写成 `锻造 @title`（文本层） |

### 2.3 按钮内部建议结构

```
Btn_Forge
├── bg @up              ← 底图
└── 锻造 @title         ← 文字（带 @title）
```

装备格：

```
Btn_EquipSlot
├── bg @up
├── Loader_Icon         ← 装备图（可换）
├── Text_Tier           ← 「1阶」
└── Text_Level          ← 「0级」
```

### 2.4 特殊后缀（写在名字里）

| 后缀 | 什么时候用 |
|------|------------|
| `@up` / `@down` | 按钮状态底图 |
| `@title` | 按钮/进度上要给程序改的标题 |
| `@icon` | 按钮/标签自带的图标槽（转成固定名 icon） |
| `@bar` | 进度条填充 |

一般可变数字用 `Text_Xxx` 即可；只有当它是「按钮标题 / 进度标题」时才加 `@title`。

## 3. Figma 操作要点

1. **画板名 = PSD 文件名**（如 `RoleEquipScreen`）  
2. 用 Frame/编组收纳区域（顶栏、装备区…），层级会保留  
3. 本地图：选中图层 → Fill → Image → 选文件；要换图的用 `Loader_` 或 `@icon`  
4. 导出 PSD 前打开 Photoshop 检查：组名还在、文字还是文字层  

导出前自检：

- [ ] 能点的都是 `Btn_`  
- [ ] 文案是 `Text_` 文本层  
- [ ] 可换图是 `Loader_`  
- [ ] 进度是 `Bar_` + `@bar`  
- [ ] 列表区是 `List_`（内容 item 另起 `Com_`/`Btn_`）  

---

## 4. 转换命令（通常程序来跑）

```sh
cd D:\Project\FairyGUI\psd2fgui
node convert 你的\RoleEquipScreen.psd
```

---

## 5. FairyGUI 里还要手做的

- `List_`：改成列表，指定 item  
- Tab 切换、动画、九宫：编辑器配置  

---

## 6. 给 Agent / 程序提需求怎么写

```
界面：RoleEquipScreen 1080×1920
参考图：（附图）

类型约定：
- 装备格可点 → Btn_EquipSlot
- 货币数字 → Text_Gold 等
- 装备图标 → Loader_
- 背包列表 → List_Bag（编辑器配 List）
- 锻造/分解 → Btn_Forge / Btn_Decompose
```
